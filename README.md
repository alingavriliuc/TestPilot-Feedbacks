# Testpilot

> ⚠️ **Early prototype.** Testpilot is in active development. Things will break — that's exactly what this repo is for. Please [open an issue](../../issues/new/choose) when they do.

**Testpilot** is a Chrome & Firefox extension that helps developers, QA engineers, and power users work with web forms faster:

- 🪄 **Autofill any form** with realistic, type-aware test data
- 🎬 **Record multi-step scenarios** (fill → click → next page → fill → submit)
- ⏯️ **Replay** scenarios with configurable inter-action delays
- 👤 **Reusable profiles** stored locally in your browser
- 📦 **Export / import** your data as JSON — nothing leaves your machine

## Reporting bugs & suggesting features

This is the main reason this repo exists. 🙏

- 🐞 **Found a bug?** → [Open a bug report](../../issues/new?template=bug_report.md). Please include browser + version, the page you were on (or a minimal repro), and what you expected vs. what happened. Screenshots/screen recordings help a lot.
- 💡 **Have an idea or feature request?** → [Open a feature request](../../issues/new?template=feature_request.md).
- ❓ **Question or general feedback?** → Use [Discussions](../../discussions).

Before opening an issue, a quick search of [existing issues](../../issues?q=) helps avoid duplicates.

## Privacy

Testpilot runs entirely in your browser. Profiles, recordings, and generated data are stored via the browser's local extension storage and never sent anywhere. Export/import is manual and goes through a JSON file you control.
